# DORM - Личный кабинет для студентов и молодых специалистов

## Описание проекта

DORM — это экосистема, которая объединяет образовательные, карьерные и социальные сервисы. Цель проекта — создать универсальный личный кабинет для пользователей (студентов, молодых специалистов, активной молодёжи), где они смогут:
- Управлять учётной записью и данными.
- Подключать мини-приложения (mini apps) для учёбы, карьеры, общения, новостей и т.д.
- Участвовать в геймификационных механиках (XP, уровни, монеты, ачивки, челленджи, сезоны).
- Настраивать интерфейс (темы, дашборды, виджеты).

## Архитектура проекта

Проект состоит из двух основных частей: фронтенд и бэкенд. Бэкенд написан на Node.js с использованием Express, а фронтенд на React.js. Для хранения данных используются MongoDB и PostgreSQL. Docker используется для контейнеризации всех компонентов приложения.

План развития проекта
MVP (Minimum Viable Product)
Ядро системы :
Регистрация/авторизация через email и соцсети.
Управление профилем.
Настройки аккаунта (выбор темы оформления, виджетов, уведомлений).
Геймификация :
Реализация системы DORM Coin и уровней.
Начисление XP за выполненные действия (например, прохождение курсов).
Добавление базовых ачивок и челленджей.
Мини-приложения :
Мини-приложение «Курсы» с базой курсов, прохождением и тестами.
Мини-приложение «Вакансии» для просмотра и откликов.
Расширение функционала
Социальные модули :
Добавление чата и ленты постов.
Реализация функций лайков, комментариев и лидербордов.
Сезонность и челленджи :
Создание сезонной системы с обновляемыми челленджами и эксклюзивными наградами.
Мобильное приложение :
Разработка мобильного приложения с использованием React Native или Flutter.
Монетизация и масштабирование
Маркетплейс мини-приложений :
Создание маркетплейса мини-приложений с возможностью установки и удаления приложений.
Локализация и глобальная интеграция :
Добавление поддержки нескольких языков и интеграций с глобальными платформами (LinkedIn, Coursera).
Блокчейн и криптовалюта :
Переход DORM Coin в статус криптовалюты с использованием блокчейна.
Критерии приемки (MVP)
Пользовательский интерфейс :
Регистрация и логин (JWT), сохранение сессии.
Страница профиля: просмотр XP/Coins, уровень, ачивки.
Мини-приложения: хотя бы одно (Курсы) должно позволять начислять XP.
Геймификация :
Начисление XP и Coins, перерасчёт уровня.
Несколько ачивок и челленджей (прогресс обновляется).
Производительность :
Загрузка основных экранов ≤ 3 сек при обычном соединении.
Безопасность :
JWT-токен проверяется на защищённых маршрутах.
Пароли хранятся в зашифрованном виде (bcrypt).
Адаптивность :
UI должен корректно отображаться на десктопе и мобильных устройствах (минимальная адаптация).
